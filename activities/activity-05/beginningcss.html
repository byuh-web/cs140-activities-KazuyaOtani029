Explain the difference between inline, embedded and external CSS

The difference is the place to write code. Inline CSS is written in HTML tag. If user would use Inline CSS, user writes style="" in HTML tag. Embedded CSS is the way to write in HTML file. This code affects only inside the head section. External css is places in the css file. In actual, this way is the most popular way due to be recyclable and avoidable being mess in the HTML file.

Explain why you should only use external CSS.

The reason why external css is the most useful way to use. Because it is recyclable and avaiable to multiple pages. Furthermore, if user wants to change an entire page, it will be finished immeaditely by using external css.
