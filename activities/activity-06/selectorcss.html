1,List out the different ways to select an item.

tag selector, class selector, ID selector, descendant selector, child selector, adjacent selector, general sibling selector, attribute selector, universal selector, Direct Child Combinator, General Sibling Combinator, Adjacent Sibling Combinator, Pseudo-classes.

2, What is the difference between an ID and a Class?

ID can be only used once. It can be used when identifying one element. Class can be used multiple times. It is useful when applying common style to multiple elements.

3, What is the difference between a SPAN tag and a DIV tag?

div behaves as a block element by default, whilst span behaves as inline by default. It is suitable to make use of these elements whenever style considerations ot other functionality requires the use of a container of some kind.

4, What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

Header, main, and footer are the semantic tag. Each of elements tells the role of them to browser or search engine.
